Отчёт по Лабораторной работе №5 
Тема "Файлы"
Выполнить все задания в одном проекте в виде статических методов одного класса.
В задании 1 и 2 бинарные файлы, содержат числовые данные, исходный файл необходимо заполнить
случайными данными, заполнение организовать отдельным методом.
В задании 3 бинарные файлы содержат величины типа struct, заполнение исходного файла
необходимо организовать отдельным методом.
В задании 4 в текстовом файле хранятся целые числа по одному в строке, исходный файл
необходимо заполнить случайными данными, заполнение организовать отдельным методом.
В задании 5 в текстовом файле хранятся целые числа по несколько в строке, исходный файл
необходимо заполнить случайными данными, заполнение организовать отдельным методом.
В задании 6 в текстовом файле хранится текст

Задание 1.Бинарные файлы
Переписать в другой файл последовательного доступа те элементы, которые кратны k.

Задание 2. Бинарные файлы
Скопировать элементы заданного файла в квадратную матрицу размером n×n (если элементов
файла недостает, заполнить оставшиеся элементы матрицы нулями). Заменить все столбцы на
столбец с минимальной суммой элементов

Задание 3. Бинарные файлы и структуры
Файл содержит сведения об игрушках: название игрушки, ее стоимость в рублях и возрастные
границы (например, игрушка может предназначаться для детей от двух до пяти лет).Определить
стоимость самого дорогого конструктора.

Задание 4. Текстовые файлы
Найти сумму тех элементов файла, которые равны своему индексу (индексацию элементов файла
в этой задаче начинать с нуля).

Задание 5. Текстовые файлы
Вычислить произведение элементов, которые кратны заданному числу k.

Задание 6. Текстовые файлы
Переписать в другой файл строки, в которых нет русских букв.

АЛГОРИТМ РЕШЕНИЯ ДЛЯ КАЖДОГО ЗАДАНИЯ:

Задание 1 (бинарные файлы, кратные числа):

1. Создать бинарный файл со случайными целыми числами
2. Запросить у пользователя число k
3. Открыть исходный файл для чтения
4. Открыть выходной файл для записи
5. Читать числа из исходного файла по одному
6. Если число кратно k - записать в выходной файл
7. Подсчитать количество найденных чисел
8. Вывести результат

Задание 2 (бинарные файлы, матрица):

1. Создать бинарный файл со случайными числами
2. Запросить размер матрицы n
3. Читать числа из файла в матрицу n×n
4. Если чисел не хватает - заполнить нулями
5. Вычислить суммы столбцов матрицы
6. Найти столбец с минимальной суммой
7. Заменить все столбцы на столбец с минимальной суммой
8. Вывести исходную и результирующую матрицы
9. Сохранить результат в файл

Задание 3 (структуры, игрушки):

1. Определить структуру Toy с полями: name, price, minAge, maxAge
2. Создать массив структур с различными игрушками
3. Записать массив в бинарный файл
4. Прочитать файл и найти конструкторы
5. Среди конструкторов найти самый дорогой
6. Вывести информацию о всех игрушках и результат

Задание 4 (текстовые файлы, сумма):

1. Создать текстовый файл с числами по одному в строке
2. Открыть файл для чтения
3. Читать числа последовательно, сохраняя индекс
4. Если число равно своему индексу - добавить к сумме
5. Вывести найденные числа и сумму
6. Сохранить результат в отдельный файл

Задание 5 (текстовые файлы, произведение):

1. Создать текстовый файл с несколькими числами в строке
2. Запросить число k у пользователя
3. Читать файл построчно, разбивая на числа
4. Для каждого числа проверять кратность k
5. Если число кратно k - умножать на произведение
6. Вывести результат произведения
7. Сохранить результат в файл

Задание 6 (текстовые файлы, строки без русских букв):

1. Создать текстовый файл с различными строками
2. Открыть исходный и выходной файлы
3. Читать строки из исходного файла
4. Проверять каждую строку на наличие русских букв
5. Если русских букв нет - копировать в выходной файл
6. Вывести исходные и отобранные строки
7. Подсчитать количество скопированных строк
